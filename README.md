# Student Grade Management System

This is a simple Java console application to manage student grades.

# Features Available:
* Add a Student - Allows adding a new student with ID, Name, and Marks.
* Display All Students - Shows a list of all registered students.
* Search for a Student by ID - Quickly find a specific student using their unique ID.
* Calculate & Display Average Mark - Calculates the overall average score of all students.
* Exit- Safely close the application.
