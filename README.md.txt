Student Grade Management System

Project Purpose
This is a simple Java console-based application developed using Object-Oriented Programming (OOP) concepts. The system allows educational institutes to digitize their student record management, moving away from manual paper records. Teachers can use this system to store, view, search, and calculate student grades efficiently.

 Features
* Add new student records (ID, Name, and Marks).
* View all existing student records.
* Search for a specific student using their unique Student ID.
* Calculate and display the average marks of all students in the system.

 How to Run the Program
1. Ensure you have Java Development Kit (JDK) installed on your computer.
2. Clone this repository or download the source files.
3. Open your terminal or command prompt in the project directory.
4. Compile the Java files:
bash
   javac Main.java Student.java


 5. Run the application:
bash
   java Main
